จากในห้องที่เรียนเรื่อง fetch API และมาต่อยอดในเรื่องของ Method เลยไปหาโจทย์จาก ChatGPT มานั่งลองฝึกดูครับ

ขั้นตอนและสิ่งที่ได้ฝึก

- fetch มาจากเว็บไซต์ที่แชทให้มา (ข้อนี้ใช้ method POST)
- แล้วนำมา Add DOM โดยกดปุ่มเพื่อส่ง JSON ที่มี { title, body } เพื่อทำการสร้าง
- แล้วหลังจากการสร้างนำมาแสดงใน div นี้ resultDiv ด้วย .textContent
- แปลงข้อมูล js (object/array) ให้กลายเป็นสตริงแบบ JSON และจัด format JSON.stringify(data, null, 2)
