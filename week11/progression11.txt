จากในห้องที่เรียนเรื่อง fetch API เลยไปหาโจทย์จาก ChatGPT มานั่งลองฝึกดูครับ

ขั้นตอนและสิ่งที่ได้ฝึก (เรียนรู้ความรู้ DOM เพิ่มเพราะรู้สึกว่าตัวเองยังมองภาพไม่ออกและยังไม่ชำนาญและนำความรู้ใหม่มาประยุกต์ด้วยเรื่อง fetch)

- fetch มาจากเว็บไซต์ที่แชทให้มา
- แล้วนำมา Add DOM เข้าไปใน li element ด้วย .textContent
- มีปุ่ม load เพื่อกดให้เนื้อหาจากที่เรา fatch มาแสดง
- ทีนี้หน้าเว็บก็จะแสดงตามที่เรา fetch ได้
